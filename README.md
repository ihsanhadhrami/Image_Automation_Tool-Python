# Image_Automation_Tool-Python
Automates batch image processing:
- Resize
- Sharpen
- Watermark
- Optimize for web

## Use Case
Ideal for marketers, e-commerce sellers, and web developers.

## How to Run
pip install -r requirements.txt
python image_automation.py

## Input
Place images inside `input_imgs/`

## Output
Optimized images saved to `output_imgs/`
